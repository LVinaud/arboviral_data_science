# data/lookup/

Tabelas de apoio versionadas no git (pequenas e estáveis). Ex.:

- `municipio_para_estacao.csv` — mapeamento município (cod_ibge) → estação automática INMET mais próxima por coordenadas.
- `codigos_municipios.csv` — códigos IBGE + nome + UF + Região de Saúde + REGIC.

Diferente de `data/raw/`, `data/interim/` e `data/processed/`, que ficam fora do git.
